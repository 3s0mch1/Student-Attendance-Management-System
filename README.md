1. Requirement Analysis
The goal of the project is to manage student attendance.
The system should allow adding students, marking attendance, and viewing attendance records.

2. System Design
The system is designed as a console-based Python program.
It uses simple menus and Python dictionaries to store data.

3. Implementation
The project is implemented using Python.
Functions are used to add students, mark attendance, and display records.

4. Testing
The program was tested by running all options and checking correct output.

5. Deployment
The program runs locally using:
python main.py

6. Maintenance
The program can be easily updated with new features in the future.
